# 🎮 NEXARDA Game Price Search

A responsive, multi-lingual React web application that allows users to search for video game prices and find the best deals using the [NEXARDA API](https://www.nexarda.com/). 

**[🔗 View Live Demo](https://lassesali.github.io/NEXARDA-game-price-search-app/)** 

## ✨ Features
* **Live Search:** Quickly find games and current pricing across multiple platforms.
* **Internationalization (i18n):** Full UI translation available in English, Spanish (Español), and Finnish (Suomi).
* **Smart State Management:** Uses `sessionStorage` to remember your search results and input history, so you don't lose your place when viewing game details.
* **Responsive Design:** Built with React-Bootstrap for a clean, mobile-friendly grid layout.

## 🛠️ Built With
* [React 18](https://react.dev/)
* [Vite](https://vitejs.dev/)
* [React Router](https://reactrouter.com/) (Routing)
* [React-Bootstrap](https://react-bootstrap.github.io/) (UI Components)
* [react-i18next](https://react.i18next.com/) (Localization)

## 🚀 Running Locally

To get a local copy up and running, follow these simple steps:

1. **Clone the repository**
   ```bash
   git clone [https://github.com/](https://github.com/)lassesali/NEXARDA-game-price-search-app.git
   ```

2. **Navigate to the project directory**

   ```bash
   cd NEXARDA-game-price-search-app
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

## 📡 API Reference
This project uses the free NEXARDA API (v3) to fetch game data and pricing information.
